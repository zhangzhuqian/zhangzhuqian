区块链的运营新在哪儿

发表于 2018-06-12 | 更新于 2018-06-12 | 分类于 币圈 |
本文共 519 字   |   阅读预计 2 分钟

运营本质上是维系产品和用户之间的连接，区块链的产品与互联网产品相比有很大的不同，这也导致了相比于互联网运营，区块链运营有很大不同，总结为一句话就是产品未成，用户先行。

区别具体体现在现有互联网行业的运营是围绕着已有产品以及由此带来的用户做运营，区块链运营则更多是围绕着共识和用户做运营，换机话说，用户也是产品的一部分。这点大家也可以从区块链招聘 JD 上看出一二，目前招聘的运营岗位基本上都是社区运营和用户运营。

有鉴于目前大部分区块链项目其实都没有落地，所以在当下这个阶段 ，产品形态不重要 甚至有没有产品不重要，运营下大力气运营的是一个共识。

这导致一个两者用户之间很大的差异，互联网行业的用户更关心产品，以及产品本身带来的价值，区块链行业的用户更关心币价&项目进度，渴望了解一切与币价相关的一切东西，但是与此同时这批用户可能根本不是项目方想要做的产品的用户。 现在币圈运营主要做的事情就是内容和社区。都是基于用户在运营，而不是产品。

##社区对于项目方的作用

  1. 用户 离项目方更近
  2. 强社区共识
  3. 社区对于币的成长越来越重要，尤其是在不以 POW 为主要共识机制的项目方中
  4. 社区自治对于用户和运营者的要求
