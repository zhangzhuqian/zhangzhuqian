区块链与比特币

简单说明

区块链是种底层协议,比特币是区块链的应用

区块链是一种新型去中心化协议，能安全地存储数据，保证信息不可伪造和篡改(篡改成本巨大)，可以自动执行设定的合约，无需任何中心化机构的审核。交易既可以是比特币这样的数字货币，也可以是债权、股权、版权等数字资产，区块链技术大大降低了现实经济的信任成本与会计成本。简单而言:区块链是一种底层协议， 比特币是在这种底层协议上开发的第一个项目，比特币本身不代表任何意义。

区块链每个节点可以做到相互信任，而不依赖任何一个中心。避免了支付宝依赖阿里巴巴,QQ依赖腾讯这种,中心化机构倒下后,整体处于不可用的状态的情况。比特币是区块链思想的第一个成功且应用较广泛的实例。

区块链衍生出来的其他应用——Sic云存储系统

Sic是一个基于区块链协议来发的去中心化的云存储系统。每个人可以将自己的计算机贡献出来，有人在Sic上存储东西，Sic会将文件分割，分别加密后存在不同的节点上，这些节点就是大家共享的电脑。 区块链协议在设计理念上假设人都是贪婪且自私的。为了鼓励大家共享电脑,在区块链之上开发系统的设计者就会设计一种以算法为基础的代币。有人贡献自己电脑的过程，系统就会奖励给这些人一些代币。

贡献的电脑有几方面作用：

第一是个证伪。区块链每个节点之所以可以做到相互信任，就是因为区块链有个全网公证的合约。每个节点都在用自己的计算机计算这条链上的每笔交易是否为真实的，大于50%，合约生效，该交易被写入区块链，并且同步任何一个节点，这里值得一说的有两点：

1、区块链伪造难度很大。 一个黑客要在区块链插入一条假数据，他的计算机的计算能力（这个叫算力）得超过全网50%。

2、节点都在全网公证，但每次只有一个人是最快的，最快的这个人能的到系统奖励的代币。而这个公证的过程就叫挖矿，公证的机器或人叫矿工。

第二是出租自己的硬盘空间。出租自己的硬盘空间,作为整个系统的存储空间,系统将给予奖励。 这也就是说每个系统都会有个代币，而Sic存储系统上的代币就是SC(Sic coin)币。

区块链存储，通过区块链终端以经济激励的方式让用户把闲置的存储资源和带宽资源贡献出来，利用这些资源为个人用户或企业提供相关产品或服务。

区块链未来的发展

当下的区块链技术，已经从金融科技转移到其它应用领域。现在大部分团队都在想用区块链颠覆现有的互联网体系，包括腾讯、谷歌这样的大公司。不过现在还有许多问题待解决，任重而道远。

关于“WannaCry”勒索病毒

其实该病毒与比特币毫无关系，之所以在广大媒体中被命名成”比特币勒索病毒”，是因为勒索者要求受害者用比特币来支付赎金，这就牵扯到了比特币本身的匿名性，导致其在洗钱，暗网中的广泛使用，这也是人性本恶的反映。

到最后不得不说一句技术本无罪，错的是利用技术作恶的人。
