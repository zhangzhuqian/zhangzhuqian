搭建独立博客

  1. 利用hexo搭建

    1. 根目录下

      npm install -g hexo-cli

    2. 新建blog文件夹

      1. hexo init
      2. npm install

    3. 修改配置文件

    4. source下新建CNAME文件,文件内输入你的域名

  2. 使用github pages

    1. 新建仓库,略
    2. hexo的配置文件中在deploy中填写git以及仓库地址
    3. 在域名服务商处填写 域名记录:https://help.github.com/articles/using-a-custom-domain-with-github-pages/

  3. blog部署到git上访问域名
